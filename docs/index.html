MathCell
a simple straightforward way to include interactive mathematics in a web page

  • Why would I want to use MathCell?

    The ubiquity now of web browsers provides an incredible opportunity to move beyond the static world of paper for the presentation of interactive mathematics. The default language of the browser is JavaScript and modern presentations use that language. MathCell provides a quick way to set up interactive elements for input into your code, as well as outputs for visualizing the results of calculations.

  • How do I get MathCell?

    MathCell is free and open source, and the code is available on GitHub. You can download the complete library to your web server and include it in the body of a page with a script tag: 

    <script src="mathcell.js"></script>
    If you do not want to download the file, a free online content delivery network can also be used to access the library: 
    <script src="https://cdn.rawgit.com/paulmasson/mathcell/master/build/mathcell.js"></script>

  • How do I start a MathCell?

    First copy this template into the body of the page:

    <div class="mathcell" style="height: 4in">
<script>

var parent = document.scripts[ document.scripts.length - 1 ].parentNode;

var id = 'id' + document.scripts.length;
parent.id = id;

MathCell( id, [ { type: '' } ] );

parent.update = function( id ) {

  var x = +document.getElementById( id + 'x' ).value;

  ( additional JavaScript here )

  var data = [];

  var config = { type: '' };

  evaluate( id, data, config );

}

parent.update( id );

</script>
</div>

    The template automatically assigns an id to each cell, which is then available to the interactive input elements that are added to the cell. This relieves the end user of the need to manually track which inputs affect which cells.

    Input elements are specified in the dictionary inside the MathCell() command. Code for processing the input is added to the body of the update function. The output element is specified in the config dictionary inside the update function.

  • What interactive inputs are available?

    There are currently three kinds of inputs: slider, buttons and checkbox. The array of inputs in the MathCell() command should contain a dictionary for each input desired, with type set equal to one of these values.

    NOTE: for inputs that are intended to provide numeric values, be sure to include the plus signs appearing here. This unary operator invokes the Number object and ensures that the value will truly be numeric.

  • What visualization outputs are available?

    There are currently four kinds of inputs: svgplot, threejs, matrix and text. The config dictionary in the update function should have its type set equal to one of these values.

  • What if I want higher-level mathematics as well?

    A pure 100% JavaScript library of special functions and common mathematical operations has been initiated and is entitled simply Math. It provides higher-level mathematics not available by default in JavaScript.
