There are currently three kinds of inputs: slider, buttons and checkbox. The array of inputs in the MathCell() command should contain a dictionary for each input desired, with type set equal to one of these values followed by a comma-separated list of optional attributes.

All inputs have two text options: a name that is used to identify the input in subsequent code, and a label that appears next to the input. The label can be any valid HTML. Single letter labels are italicized by default.

A slider input has four numeric options with default values

min0
max1
step.01
defaultmin

and the variable is assigned in code using

var name = +document.getElementById( id + 'name' ).value;

A buttons input has two additional options, one an array and the other numeric, with default values

list[1,2,3]
defaultlist[0]

and the variable is assigned in code using

var name = +document.querySelector( 'input[name=' + id + 'name]:checked' ).value;

A checkbox input has a single additional option default, which can be set to either 'checked' or true to have the box initially checked. The variable is assigned in code using

var name = document.getElementById( id + 'name' ).checked;

IMPORTANT: for inputs that are intended to provide numeric values, be sure to include the plus signs appearing here. This unary operator invokes the Number object and ensures that the value will truly be numeric.
