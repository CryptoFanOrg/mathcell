There are currently four kinds of outputs: svgplot, threejs, matrix and text. The config dictionary in the update function should have its type set equal to one of these values.


An svgplot output displays a two-dimensional plot of the data. The input data should be an array of arrays of data points, i.e. a three-dimensional array. An svgplot has options and default values

xMincalculated
xMaxcalculated
yMincalculated
yMaxcalculated
axestrue
ticksauto
axesLabelsnone
includeOriginfalse
includeVerticalsfalse
equalAspectfalse

ticks can be set to an array with two numbers determining distances between ticks. colors is an array specifying the color of each array in the input data, with each given by standard HTML color codes.


A threejs output displays a three-dimensional plot of the input data with the options and default values

xMincalculated
xMaxcalculated
yMincalculated
yMaxcalculated
zMincalculated
zMaxcalculated
aspectRatio[1,1,1]
axesfalse
axesLabels['x','y','z']
decimals2
frametrue
viewpointauto
ambientLightrgb(127,127,127)

A matrix output displays the contents of a two-dimensional array in matrix form. You may need to adjust the width and height of the MathCell to show the entire array properly.


A text output displays the result of a calculation in raw form. The data for this output can be a number, a string or an array as desired. The output has one option and default

centerfalse
