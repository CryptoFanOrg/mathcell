Options for graphics objects are specified as dictionaries. Color and opacity can be omitted, with default values of blue and one, respectively.


arrow( begin, end, color ) — two-dimensional arrow from beginning to end

box( width, depth, height, options ) — box of specified dimensions centered at the origin

cylinder( radius, height, options ) — cylinder of specified dimensions centered at the origin; an option of steps determines the smoothness of the surface, an option of axis the arbitrary direction of the cylinder, and an option of center the position of the cylinder

line( points, options ) — two- or three-dimensional line joining an array of points; an option of radius draws the line as an extended cylinder, and an option of endcaps equal to true includes spheres to smooth the joints

sphere( radius, options ) — sphere of specified radius centered at the origin; an option of steps determines the smoothness of the surface

text( string, point, color, fontsize ) — string of text at the two- or three-dimensional point, with default color of black and font size of 14
