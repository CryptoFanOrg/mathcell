Color and opacity can be omitted in these functions, with default values of blue and one, respectively.


plot( f, range, color ) — plot a function over a range given as an array [begin,end] or [begin,end,points]

listPlot( points, color ) — plot an array of points

parametric( vector, xRange, yRange, color, opacity ) — plot a vector function of two variables over ranges given as arrays [begin,end] or [begin,end,points]


isoline( f, xRange, yRange, color ) — implicit line plot of a function of two variables over ranges given as arrays [begin,end] or [begin,end,points]

isoband( f, xRange, yRange, color ) — implicit filled contour plot of a function of two variables over ranges given as arrays [begin,end] or [begin,end,points]

isosurface( f, xRange, yRange, zRange, color, opacity ) — implicit surface plot of a function of three variables over ranges given as arrays [begin,end] or [begin,end,points]
